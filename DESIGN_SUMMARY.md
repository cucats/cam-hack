# Cam Hack 2025 - New Splash Page Design

## Design Overview

A stunning, futuristic splash page featuring Three.js animations, neon green aesthetics, and immersive parallax effects.

## Color Palette

- **Primary Green**: #00ff88 (neon green for titles and accents)
- **Secondary Green**: #00ffcc (cyan-green for highlights)
- **Deep Green**: #0d7f4a (for subtle elements)
- **Dark Background**: #0a0e0d (almost black with green tint)
- **Light Text**: #e8fff5 (very light green tint)

## Typography

- **Headings**: Space Grotesk (modern, tech-inspired)
- **Body**: Inter (clean and readable)
- **Numbers/Countdown**: Orbitron (futuristic, digital aesthetic)

- 3000 particle system with three color variants

## Animations

- **Particle System**: Continuous rotation and wave motion
- **Geometric Shapes**: Multi-axis rotation
- **Hero Logo**: Floating animation (3s infinite)
- **Countdown**: Glowing pulse effect
- **Cards**: Transform on hover with glow effects
- **Parallax**: Different scroll speeds for depth
- **Gradient Borders**: Rotating gradients on featured cards
- **Navigation**: Underline slide-in animation
- **Buttons**: Scale and glow on hover

## Responsive Design

- Mobile-first approach
- Hamburger menu for mobile devices
- Flexible grid layouts
- Adjusted font sizes for mobile
- Touch-friendly button sizes

## Technical Implementation

- **Framework**: SvelteKit 5
- **3D Graphics**: Three.js
- **Styling**: Tailwind CSS + Custom CSS
- **Animations**: CSS animations + Three.js
- **Performance**: Optimized particle count and render loop

## Removed Elements

- Shared footer (as requested)

- Static backgrounds

## Browser Compatibility

- Modern browsers with WebGL support
- Fallback for devices without 3D acceleration
- Responsive across all screen sizes
